准备 Mysql 配置文件(my.cnf)

my.cnf 配置说明

  1. default_authentication_plugin mysql8.0 引入了新特性 caching_sha2_password,这种密码加密方式 navicat 不支持,这里设置为 mysql_native_password
  2. server-id 在 mysql 集群中需要唯一
  3. read-only 1:表示该库只读/0:表示该库为读写 (默认 root 用户无视此规则,super-read-only 可以配置 root 的读写权限)
  4. binlog_format 二进制日志格式:
    • ROW : 记录一行数据的变更
    • STATEMENT : 记录数据操作的对应 Sql 语句
    • MIXED : ROW 和 STATEMENT 的混合(当 STATEMENT 有歧义时会转换为 ROW 格式)

创建一个自定义 Docker 网络

因为 docker 容器的 ip 是变化的,自定义网络,可以为每个容器指定 IP,搭建主从的时候会方便一些
–driver bridge 使用桥接模式
–subnet 172.32.0.0/16 表示子网 ip 可以分配 172.32.0.2 到 172.32.255.255
–gateway 172.32.0.1 网关
